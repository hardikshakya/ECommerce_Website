# ECommerce Website

## Requirements

-   [Node v10.15+](https://nodejs.org/en/download/current/)
-   [Express](https://expressjs.com/)

## Project Setup

1. Clone the repository

2. Then go to the project folder and run following command in command line,
```bash
nvm use
npm install
```

## How To Run
```bash
npm start
```
OR
```bash
node app.js
```
